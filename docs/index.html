Avgiftskutt på drivstoff er truleg ulovleg

Finansminister Jens Stoltenberg seier at Stortinget sitt vedtak om å redusere CO₂-avgiften på drivstoff troleg er ulovleg. Det er fordi det kan vere statsstøtte, som ikkje er tillate. Statsstøtte er når staten gir pengar eller fordeler til bedrifter på ein måte som ikkje er rettferdig. Stoltenberg seier at dette kan få store konsekvensar for næringslivet. Drivstoff blir brukt til å få biler, bussar og lastebiler til å køyre. Når avgiften blir sett ned, betaler folk mindre for bensinen. Men ifølgje finansministeren kan staten få bota frå andre land for dette.

📖 Visste du at…?
CO₂-avgift: Ein skatt på drivstoff for å gjere det dyrare å forureina miljøet
Statsstøtte: Når staten gir pengar eller fordeler til bedrifter

Mann sikta for åtte overfall av kvinner

Værfenomenet El Niño kan kome i år

Verdens meteorologiorganisasjon (WMO) advarer om at ein kraftig versjon av værfenomenet El Niño kan utvikle seg i år. El Niño er når det blir uvanleg varmt på havet, og det kan påverke været over store deler av verden. Det kan føre til hetebølgjer, tørke og kraftige regnfall. Folk over heile verden må kunne vere klare for uventa vêr. El Niño skjer naturleg, men det skjer oftare no på grunn av klimaendringane. Det er viktig at land planlegg godt og forbereder seg på møgelege problem.

📖 Visste du at…?
El Niño: Ein værfenomen som gjer det uvanleg varmt på havet
Meteorologiorganisasjon: Ein organisasjon som studerar vêr og klima
